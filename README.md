# Login Forms Website

This is a simple website showcasing login forms.

## Usage

Open `index.html` in a web browser to view the login form.

## Files

- `index.html`: Main login page
- `assets/HTML/news.html`: News page
- `assets/CSS/`: Stylesheets
- `assets/JS/`: JavaScript files
- `assets/IMG/`: Images
- `assets/fonts/`: Fonts

## Troubleshooting

Ensure all asset paths are correct.
